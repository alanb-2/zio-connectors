# zio-connectors

A library containing pre-curated connector components for use with ZIO code.

## Test

### Unit

```shell
sbt clean test
```

### Coverage report

1. Run the test with coverage analysis
    ```shell
    sbt clean coverage test
    ```
2. Generate the report
    ```shell
    sbt coverageReport
    ```

## Publish

```shell
sbt clean publishLocal
```
